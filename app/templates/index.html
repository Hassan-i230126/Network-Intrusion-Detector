About NSL-KDD

The NSL-KDD dataset is an improved version of the KDD Cup 1999 dataset, widely used for benchmarking network intrusion detection systems. It addresses key issues of the original dataset: redundant records were removed to prevent classifier bias, and the training and test sets were carefully designed to avoid overlap.

The dataset contains 41 features describing network connection attributes across four categories: basic features (e.g., duration, protocol), content features (e.g., number of failed logins), time-based traffic features (e.g., connection rates), and host-based traffic features. Traffic is labeled as either normal or one of four attack categories: DoS (denial-of-service), Probe (surveillance/scanning), R2L (remote-to-local unauthorized access), and U2R (user-to-root privilege escalation).
